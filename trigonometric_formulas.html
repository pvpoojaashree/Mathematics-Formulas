Trigonometry Identities

Tangent Identities

  • tanθ=sinθ/cosθ
  • cotθ=cosθ/sinθ

Even And Odd Identities

  • sin(-θ) = -sinθ
  • cos(-θ) = cosθ
  • tan(-θ) = -tanθ
  • cos(-θ) = -cosθ
  • sec(-θ) = secθ
  • cot(-θ) = -cotθ

Reciprocal Identities

  • cosecθ = 1/sinθ
  • sinθ = 1/cosecθ
  • secθ = 1/cosθ
  • cosθ = 1/secθ
  • cotθ = 1/tanθ
  • tanθ = 1/cotθ

Double Angle Identities

  • sin(2θ) = 2sinθcosθ
  • cos(2θ) = 2cos2θ - 1
  • tan(2θ) = 2 tanθ/1 - tan2θ

Pythagorean Identities

  • sin2θ + cos2θ = 1
  • tan2θ + 1 = sec2θ
  • cot2θ + 1 = cosec2θ

Cofunction Identities

  • sin(Π/2-θ) = cosθ
  • cosec(Π/2-θ) = secθ
  • tan(Π/2-θ) = Cotθ
  • cos(Π/2-θ) = sinθ
  • sec(Π/2-θ) = cosecθ
  • cot(Π/2-θ) = tanθ

Trignometric Formulas

Angle Addition Formulas

  • sin (A + B) = sinA cosB + cosA sinB
  • sin (A - B) = sinA cosB - cosA sinB
  • cos (A + B) = cosA cosB - sinA sinB
  • cos (A - B) = cosA cosB + sinA sinB
  • tan (A + B) = tanA + tanB/1 - tanA tanB
  • tan (A - B) = tanA - tanB/1 + tanA tanB
  • cot (A + B) = cotA cotB - 1 / cotA + cotB
  • cot (A - B) = cotA cotB + 1 / cotB - cotA

Sum To Product Formulas

  • sinA + sinB = 2.sin(A + B / 2).cos(A - B / 2)
  • sinA - sinB = 2.sin(A - B / 2).cos(A + B / 2)
  • cosA + cosB = 2.cos(A + B / 2).cos(A - B /2)
  • cosA - cosB = -2.sin(A + B / 2).sin(A - B / 2)

Law Of Sines

  • a / sin A = b / sin B = c / sin C

Law Of Cosines

  • a2 = b2 + c2 - 2bc cosA
  • b2 = a2 + c2 - 2ac cosB
  • c2 = a2 + b2 - 2ab cosC

Law Of Tangents

  • a - b / a + b = tan[½(A - B)]/tan[½(A + B)]

Important Formulas

  • sin2A = 2sinAcosA
  • cos2A = cos2A - sin2A
  • tan2A = 2tanA / 1 - tan2A
  • sin2A - sin2B = sin(A + B) sin(A - B)
  • cos2A - sin2B = cos(A + B) cos(A - B)
  • cos2A - cos2B = -sin(A + B) sin(A - B)
  • 2sinAcosB = sin(A + B) + sin(A - B)
  • 2cosAsinB = sin(A + B) - sin(A - B)
  • 2cosAcosB = cos(A + B) + cos(A - B)
  • 2sinAsinB = cos(A-B)-cos(A+B)
